# Blogging_Using_Node
Use Mongoose Version 5.13.7 Else You Might Face Error!
First Run Command $npm init
When you get all the necessary modules,
then Install some of the modules like EXPRESS, MONGOOSE, MORGAN, EJS , NODEMON...
Write this and every module will be installed in one line
$npm i express mongoose@5.13.7 morgan ejs nodemon
This is it and you are done with the environment!
Now run the server and edit the blog the way you want!
